# LeadFlow CRM

A modern, full-stack Customer Relationship Management (CRM) application designed to optimize sales pipelines, track client interactions, and provide real-time performance analytics.

🔗 **Live Demo:** [https://leadflow-clients.onrender.com](https://leadflow-clients.onrender.com)

---

## 🚀 Core Features

* **Interactive Business Dashboard:** Tracks total leads, active client metrics, and visual performance charts.
* **Pipeline Management:** Stages leads dynamically through custom workflows (New, Contacted, Won, Lost).
* **Live Activity Feed:** Centralized audit logs capturing data creation and status changes instantly.
* **Responsive Layout:** Adaptive user experience optimized for both mobile and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Tailwind CSS, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Hosting:** Render (Full-Stack Continuous Deployment)
